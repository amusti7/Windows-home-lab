# Windows-home-lab
## Objective
Built a Windows home lab using VirtualBox to practice networking, troubleshooting, and Windows administration skills relevant to IT support and help desk roles.

---

# Networking Commands Lab

## Commands Used
- ipconfig
- ping
- nslookup
- tracert

## What I Learned
- How to identify IP configuration
- How to test network connectivity
- How DNS resolution works
- How to trace network routes

---

# User Account Management Lab

## Tasks Completed
- Created standard and administrator accounts
- Practiced password management
- Explored Windows permissions and User Account Control (UAC)

## What I Learned
- Difference between standard and admin users
- Importance of least privilege access
- Basic Windows account management

---

# Network Troubleshooting Lab

## Scenario
Simulated internet connectivity failure by disabling the network adapter.

## Troubleshooting Steps
1. Tested connectivity using ping
2. Verified network adapter status
3. Re-enabled network adapter
4. Confirmed restored internet connectivity

## Resolution
Internet connectivity restored successfully.
